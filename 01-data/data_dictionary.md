# Data Dictionary — Bank Customer Churn

| Column | Type | Description |
|--------|------|-------------|
| CustomerId | int | Unique customer identifier |
| Surname | string | Customer surname |
| CreditScore | int | Credit score (350–850) |
| Geography | string | Country: France, Germany, Spain |
| Gender | string | Male / Female |
| Age | int | Customer age |
| Tenure | int | Years as customer (0–10) |
| Balance | float | Account balance |
| NumOfProducts | int | Number of bank products held (1–4) |
| HasCrCard | int | Has credit card: 1=Yes, 0=No |
| IsActiveMember | int | Active member: 1=Yes, 0=No |
| EstimatedSalary | float | Estimated annual salary |
| Exited | int | **Target variable** — Churned: 1=Yes, 0=No |
